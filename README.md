# Aritra Saha Portfolio

Modern static portfolio website built with HTML, CSS, and vanilla JavaScript.

## Files

- `index.html` — page structure and portfolio content
- `styles.css` — responsive UI, dark/light theme, animations
- `script.js` — typing effect, scroll reveals, menu, theme toggle, cursor effects

## Run locally

Open `index.html` directly in a browser, or serve the folder with any static server.
